# Consultant Staffing & Capacity Planner
### Overview 
Consultant Staffing & Capacity Planner is a web application designed to help project managers allocate consultants to projects based on skills, availability, and workload capacity.
The platform provides a centralized view of consultants, projects, assignments, and utilization metrics, enabling managers to make informed staffing decisions and avoid resource bottlenecks.
This project was built as an enterprise-style application inspired by challenges commonly faced by consulting firms and professional services organizations.
